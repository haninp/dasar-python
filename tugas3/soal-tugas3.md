## Tipe Data
```
>>> x = 5
>>> y = "Type Data"
>>> print("type data x adalah =",type(x))
<class 'int'>
>>> print("type data y adalah =",type(y))
<class 'str'>
```
## Convert Tipe Data
### convert string to int
```
x = "5"
print(type(int(x)))
```

### convert int to string
```
y = 5
print(type(str(y)))
```
